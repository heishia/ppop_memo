# 사용 사례

## 메모 생성
1. 앱 실행 시 자동으로 새 메모 창 열기
2. 제목과 내용 입력
3. 자동 저장

## 모드 전환
1. Ctrl+T 또는 모드 변경 버튼 클릭
2. 텍스트 모드 ↔ 캔버스 모드 전환
3. 모드 상태 저장

## 판서 기능 사용
1. 캔버스 모드로 전환
2. Canvas에 그림 그리기
3. Handwriting Recognition 실행
4. 신뢰도가 기준값 이상이면 텍스트로 변환
5. 변환된 텍스트는 에디터에 추가

## 파일 열기
1. txt/md 파일 더블클릭
2. 앱에서 파일 내용을 새 메모로 열기

## 검색
1. 검색어 입력
2. 제목과 내용에서 검색
3. 검색 결과 표시
