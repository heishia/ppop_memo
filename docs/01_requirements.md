# 요구사항

## 기능 요구사항

### 메모 관리
- 메모 생성, 수정, 삭제
- 제목과 내용 저장
- 자동 저장

### 스티키 메모
- 여러 메모 창 동시 표시
- 윈도우 위치/크기 저장 및 복원
- 최상단 고정 기능

### 모드 전환
- 텍스트 모드 ↔ 캔버스 모드
- 단축키: Ctrl+T
- 모드 변경 버튼

### 파일 연결
- txt/md 파일 연결 프로그램 등록
- 파일 열기 시 새 메모 생성

### 폴더 분류
- 계층적 폴더 구조
- 메모 이동 기능

### 검색
- 제목 + 텍스트 내용 검색
- 실시간 검색

### 판서 기능
- Canvas에 자유롭게 그리기
- Handwriting Recognition
- 신뢰도 기반 텍스트 변환

## 비기능 요구사항

- 로컬에서만 작동 (서버 불필요)
- 빠른 시작
- 직관적인 UI
